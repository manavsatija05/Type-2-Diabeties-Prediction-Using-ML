# Type-2-Diabeties-Prediction-Using-ML
This project aims to predict the likelihood of Type 2 Diabetes using various Machine Learning algorithms based on medical diagnostic data. The model analyzes several health-related parameters and compares different ML algorithms to determine the most accurate prediction model.

The project demonstrates:
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Model training and testing
Performance comparison of ML algorithms
Diabetes prediction system implementation
